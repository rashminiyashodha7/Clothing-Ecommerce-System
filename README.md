# Clothing-Ecommerce-System
User-centered clothing e-commerce platform for seamless online shopping and improved customer experience.

This project is a full-stack clothing e-commerce platform designed to help small businesses establish a user-friendly digital storefront. The system focuses on enhancing customer engagement through intuitive UI design, secure transactions, and smooth navigation.

It includes features such as product browsing, cart management, and payment integration using Stripe API. The platform was developed with a strong focus on aligning user experience with business goals, ensuring both usability and functionality.

Tech Stack: React, Node.js, JavaScript, HTML, CSS, SQL, Stripe API, Postman
